Learn_Laravel
====

Kotaro AraiのLaravel学習用リポジトリです。

## Description
ToDoの登録・更新・削除ができます。

言語:PHP7.2
フレームワーク:Laravel
データベース：SQLite
デザイン：BootStrap

ローカルの環境構築:docker
使用コンテナ:Laradock

デプロイ:AWS EC2
(ECSは不使用)
